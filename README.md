# VM汉化组文档站

立即访问：<https://docs.vmct-cn.top/>

## 📖 使用与部署

我们推荐使用 VSCode 编辑器进行开发。

在开发前，请先安装 [NodeJS](https://nodejs.org/zh-cn/download/prebuilt-installer)，推荐安装 Node24。

安装依赖：

```sh
pnpm i
```

开发预览：

```sh
pnpm run docs:dev
```

构建：

```sh
pnpm run docs:build
```
